Para lograr el funcionamiento del programa se deben seguir el siguiente manual:
1. Descargar y descomprimir el archivo
2. El archivo obligatoriamente debe estar insertado en la carpeta 'HTDOCS' de 'XAMPP'
3. Se debe insertar la BDD llamada 'administadores' dentro de 'PHPMyAdmin'
4. Una vez ingresado a 'localhost', seguir la ruta BAVigila/Administradores/Login/index_hurto.html.
En este archivo se verá un login y debajo un boton para el recupero de contraseña el cual al clickear lleva a un espacio para ingresar tu correo electronico.
5. Ingresar tu correo electronico y revisar la bandeja de entrada donde estara un mail de la cuenta bavigila con un link para el recupero
6. Verá una interfaz para restablecer la contraseña, ingrese la nueva contraseña e inicie sesion con su nueva contraseña.
